Challenge #3 - AI and Images

Instructions

I've collected and generated some images of classrooms for you. They are presented in a gallery below. Some of them are real images taken with a camera, and others are either completely AI generated or significanlty modified by AI. To solve this challenge, you need to identify which images are real and which are AI generated/modified. If you get stuck, there are some hints below the image gallery.

Once you've determined which images are real, you can use the first letter of each image's filename to generate a url to the next challenge:

  1. Put the real images in alphabetical order by filename (they are already in alphabetical order in the gallery).
  2. Take the first letter of each real image's filename and concatenate them together to get a string of letters. For example if class43.jpg and room_big.jpg and y_am_i_doing_this.jpg were the only three real images, then string would be "cry".
  3. Generate a bit.ly link using the string of letters you generated. For example, if your string was "cry", then you would generate a bit.ly link like: https://bit.ly/cry

Some tips:

  • You can use any AI tool you like to help you with this challenge. Some tools are better at image analysis than others. Here are a few suggestions:
    • Your eyeballs - If you've explored AI generated images before, you're own eyes will probably be able to pick out some of them.
    • WasItAI
    • ZeroGPT AI Image Detector
    • Sightengine - Requires a free account.
  • Triangulate! Don't just use a single tool. Often the tools above will contradict each other.
  • Text is often an obvious giveaway. As of Feb 2026, text in AI-generated images is often illegible or nonesense.
  • Lighting! Last year. This year, AI has become insanely good at generating realistic lighting, shadows and reflections, so don't get too distracted by lighting or you'll go insane.
  • Some other ideas: filenames, image metadata, and ask your LLM what else to look for! You can even drop the image into your LLM and ask it what it thinks.
  • If you get stuck, there are some hints below the image gallery.
